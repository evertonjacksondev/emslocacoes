## Installation

To install it and run it in your machine, do the following:

From your command line:

## Clone this repository
$ git clone https://github.com/evertonjacksondev/emslocacoes/

## Go into the repository
$ cd emslocacoes

## Install dependencies
$ npm install

## Run the app
$ npm start
